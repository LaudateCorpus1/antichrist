## Development
Development is done ith [yeoman][2]. You need to install the following:

* [Node.js][1] (`brew install nodejs`)
* [yeoman][2] (`npm install -g yo`)
* [grunt][3] (`npm install -g grunt-cli`)
* [bower][4] (`npm install -g bower`)

### Setup

	npm install
	bower install

### Dev Webserver

	grunt server

### Release build

	grunt build

### Testing

	grunt test

[1]: http://nodejs.org/
[2]: http://yeoman.io/
[3]: http://gruntjs.com/
